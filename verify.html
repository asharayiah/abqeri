Verify Downloads

Every official release is signed and published with SHA-256 checksums. Verify both before use.

1) Import Abqeri PGP public key

# Save our public key (replace with real key later)curl -sS https://downloads.abqeri.com/keys/abqeri_signing.pub | gpg --import# or fetch from keys.openpgp.org once published

2) Verify checksum

# Mac/Linuxshasum -a 256 FILENAME > local.sha256 && diff local.sha256 FILENAME.sha256# Windows (PowerShell)Get-FileHash .\FILENAME -Algorithm SHA256 | Format-List

3) Verify signature

# Signature verificationgpg --verify FILENAME.sig FILENAME# Expect: "Good signature from Abqeri Releases <releases@abqeri.com>"
