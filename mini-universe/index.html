Mini‑Universe Simulator

A reproducible black‑box sandbox for foundational physics — wave–particle duality, entanglement, the observer, vacuum↔matter, dark sectors, and black‑hole information. Deterministic by seed; identical outputs across machines.

Download Demo Package (ZIP) Quickstart (PDF) Methods Note (PDF) Signed Seeds 1‑page Intro (PDF)

How it works

The engine is a closed binary (black‑box) combined with open notebooks and scripts. Researchers run experiments with a seed and parameters; the engine emits raw CSV event logs and plots. With the same engine version, seed, and parameters, the results are bit‑for‑bit identical.

Run in 1 command

Docker (recommended):

docker run --rm -v $(pwd)/out:/out mini_universe:latest   --seed 42 --experiment bell_chsh

Outputs:

  • out/events_bell_seed42.csv
  • out/plots/bell_chsh_seed42.png

What’s included

  • Closed engine binary (no source) / Docker entrypoint
  • Open experiment layer
  • Methods & ethics note
  • Signed seed registry & checksums
  • Quickstart guide

Use‑case example

Run Bell/CHSH with a fixed seed. Analyze the event log and observe CHSH violation (e.g., S≈2.5). Lower the duality parameter to approach classical correlations (S≈2). Identical results across machines validate reproducibility.
